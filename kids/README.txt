First Five — kids' cartoon faces.

Each kid has a "face set" of six expressions, named <set>_<feeling>.png:
  k1_content.png  k1_happy.png  k1_tired.png  k1_surprised.png  k1_sad.png  k1_angry.png
  k2_*  k3_*  k4_*   (one set per kid)

In Settings > First Five, each kid picks which face set (k1-k4) is theirs.
The widget uses these for both the avatar (shows their current mood, resting
on "content") and the six tappable mood buttons. Square images look best.

To add another kid's faces, drop a new set k5_<feeling>.png ... then add a kid.
