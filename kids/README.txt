First Five widget assets.

KID FACES — each kid has a "face set" of six expressions, <set>_<feeling>.png:
  k1_content.png  k1_happy.png  k1_tired.png  k1_surprised.png  k1_sad.png  k1_angry.png
  k2_*  k3_*  k4_*   (one set per kid)
In Settings > First Five each kid picks which set (k1-k4) is theirs. The widget
uses these for the avatar (shows their current mood, resting on "content") and
the tap-to-open mood menu. To add another kid: drop k5_<feeling>.png and add a kid.

ROUTINE ICONS — the five morning steps, ff_<id>.png:
  ff_breakfast.png  ff_dressed.png  ff_hair.png  ff_teeth.png  ff_shoes.png

Square images look best; missing files fall back to an emoji.
